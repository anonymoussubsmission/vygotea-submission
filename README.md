# VygotEA — Evaluation

## Requirements
Python 3.10+, scikit-learn 1.4, faiss-cpu 1.7

## How to run
1. Upload `vygotea_config.json` to your Colab session
2. Open `VygotEA_notebook_v3.ipynb` in Google Colab
3. Run all cells in order

## Files
- `vygotea_config.json`: all experimental parameters
- `VygotEA_notebook_v3.ipynb`: full evaluation pipeline
